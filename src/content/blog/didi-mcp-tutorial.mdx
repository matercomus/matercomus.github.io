I'm excited to share that my first video tutorial was featured on the [DiDi MCP website](http://mcp.didichuxing.com/)! 

## What I Built

I created a simple demo showcasing how to use DiDi MCP (Model Context Protocol) to order taxis directly from within a custom LibreChat deployment. The demo demonstrates the power of integrating MCP servers with chat interfaces to create seamless, real-world functionality.
